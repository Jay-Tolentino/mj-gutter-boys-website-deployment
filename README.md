# MJ Gutter Boys Website Deployment (Production Case Study)

## Overview

This project documents the deployment and configuration of a live business website for MJ Gutter Boys.

Rather than focusing on front-end development, this case study highlights real-world infrastructure responsibilities including domain configuration, hosting setup, SSL security, and email deliverability troubleshooting.

Live Site: https://mjgutterboys.com

---

## Responsibilities

- Registered and managed domain via Namecheap
- Configured DNS records to point domain to Hostinger hosting
- Deployed and configured WordPress environment
- Enabled and verified SSL certificate (HTTPS)
- Customized theme, branding, and business content
- Implemented lead capture contact form
- Diagnosed and resolved email deliverability issues
- Integrated authenticated third-party mailing service using API credentials to prevent spam filtering
- Ensured reliable routing of client inquiries to business inbox

---

## Technical Work Performed

### Domain & DNS Configuration
- Connected Namecheap domain to Hostinger hosting
- Managed nameserver configuration
- Verified domain propagation

### Hosting & Deployment
- Provisioned WordPress instance on Hostinger
- Configured production environment
- Managed updates and content deployment

### SSL & Security
- Enabled HTTPS with SSL certificate
- Verified secure site connection
- Ensured encrypted client form submissions

### Email Deliverability Troubleshooting
Initial contact form submissions were being flagged as spam.

Resolution steps:
- Identified lack of email authentication
- Integrated third-party mailing service
- Configured API access key
- Authenticated outbound mail routing
- Verified successful inbox delivery

Successfully deployed a production-ready business website with secure HTTPS access and reliable client inquiry routing. The solution improved email deliverability and ensured uninterrupted lead capture functionality.
